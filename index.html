Preguntas sobre Bases de Datos

1. ¿Qué es un Dato?

Un dato es una representación simbólica de una entidad que permite describir, medir, o catalogar un aspecto de la realidad.

2. ¿Qué es información?

La información es un conjunto de datos procesados y organizados de manera que tienen un significado y son útiles para la toma de decisiones.

3. ¿Diferencia entre dato e información?

Un dato es una unidad básica sin significado inherente hasta que es procesada. La información es el resultado de procesar y organizar datos de tal manera que adquieren significado y contexto.

4. ¿Tipos de datos?

Los tipos de datos incluyen:

  • Numéricos (enteros, flotantes)
  • Texto (cadenas de caracteres)
  • Fecha y hora
  • Booleanos (verdadero/falso)
  • Binarios (imágenes, archivos)

5. ¿Qué es un modelado de Datos?

El modelado de datos es el proceso de crear un modelo visual que representa las estructuras de datos y las relaciones entre ellos, ayudando a diseñar y organizar bases de datos.

6. Tipos de bases de Datos, según:

Variabilidad

Estáticas (datos no cambian) y dinámicas (datos se actualizan).

Contenido

Bibliográficas, multimedia, de texto completo.

Modelo de base de datos

Relacionales, NoSQL, orientadas a objetos, entre otras.

7. ¿Qué es el modelo relacional?

El modelo relacional es un enfoque para gestionar datos donde estos se organizan en tablas relacionadas entre sí mediante llaves primarias y foráneas.

8. Describa los elementos del Diagrama entidad Relacional

Elementos incluyen entidades, atributos y relaciones. Representan objetos del mundo real y las asociaciones entre ellos.

9. ¿Qué es una entidad?

Una entidad es un objeto distinguible que tiene existencia propia en el contexto del sistema, como una persona, lugar o cosa.

10. ¿Qué es un atributo y sus tipos?

Un atributo es una propiedad o característica de una entidad. Tipos incluyen simples, compuestos, derivados y multivaluados.

11. ¿Qué es una relación?

Una relación es una asociación entre dos o más entidades. Describe cómo las entidades interactúan entre sí.

12. ¿Qué son las llaves primarias, foránea y sus características?

Llave primaria: Un atributo único que identifica de manera unívoca una entidad en una tabla.

Llave foránea: Un atributo en una tabla que hace referencia a la llave primaria de otra tabla, estableciendo una relación entre ellas.

13. ¿Qué son las reglas del Negocio?

Las reglas del negocio son restricciones y condiciones que definen las operaciones, definiciones y restricciones dentro de una organización y su sistema de información.

14. ¿Qué es la cardinalidad o mapeo?

La cardinalidad define el número de instancias de una entidad que pueden asociarse con instancias de otra entidad en una relación. Puede ser uno a uno, uno a muchos, o muchos a muchos.
